# Navigation — Deep Q-Network (DQN)

Project 1 of the [Udacity Deep Reinforcement Learning Nanodegree](https://www.udacity.com/course/deep-reinforcement-learning-nanodegree--nd893).
An agent is trained with **Deep Q-Learning** to navigate a large square world and collect bananas.

![Banana environment](https://video.udacity-data.com/topher/2018/June/5b1ab4b0_banana/banana.gif)

## The Environment

The agent moves through a square world filled with yellow and blue bananas.

- **Reward:** `+1` for collecting a yellow banana, `-1` for a blue banana. The goal is to collect as many yellow bananas as possible while avoiding blue ones.
- **State space:** `37` continuous dimensions — the agent's velocity plus a ray-based perception of objects in its forward direction.
- **Action space:** `4` discrete actions:
  | Action | Meaning |
  |:------:|:--------|
  | `0` | move forward |
  | `1` | move backward |
  | `2` | turn left |
  | `3` | turn right |
- **Solved when:** the agent achieves an **average score of +13 over 100 consecutive episodes**.

> This environment is provided by Udacity and is **similar to, but not identical to**, the Banana Collector environment on the Unity ML-Agents GitHub page. The submission must use the Udacity-provided environment below.

## Getting Started

### 1. Set up the Python environment

This project depends on the legacy Udacity DRLND stack (Python 3.6, PyTorch 0.4, Unity ML-Agents v0.4). Follow the dependency instructions in the [Udacity `Value-based-methods` repo](https://github.com/udacity/Value-based-methods#dependencies), or:

```bash
conda create -n drlnd python=3.6
conda activate drlnd
pip install -r requirements.txt
```

### 2. Download the Unity environment

Download the build that matches your OS (you do **not** need to install Unity):

- **Linux:** https://s3-us-west-1.amazonaws.com/udacity-drlnd/P1/Banana/Banana_Linux.zip
- **Mac OSX:** https://s3-us-west-1.amazonaws.com/udacity-drlnd/P1/Banana/Banana.app.zip
- **Windows (32-bit):** https://s3-us-west-1.amazonaws.com/udacity-drlnd/P1/Banana/Banana_Windows_x86.zip
- **Windows (64-bit):** https://s3-us-west-1.amazonaws.com/udacity-drlnd/P1/Banana/Banana_Windows_x86_64.zip
- **Headless (AWS / no display):** https://s3-us-west-1.amazonaws.com/udacity-drlnd/P1/Banana/Banana_Linux_NoVis.zip

Unzip it into the root of this repository and point `file_name` at the executable in the notebook, e.g. `Banana.app` (Mac) or `Banana_Linux/Banana.x86_64` (Linux).

> If you train in the **Udacity Workspace**, the environment is already present at `/data/Banana_Linux_NoVis/Banana.x86_64` — enable **GPU** before training.

## Files

| File | Description |
|:-----|:------------|
| `Navigation.ipynb` | Main notebook: starts the environment, trains the agent, plots scores, and watches the trained agent. |
| `dqn_agent.py` | The `Agent` (DQN with experience replay + fixed Q-targets) and `ReplayBuffer`. |
| `model.py` | The `QNetwork` — the PyTorch neural network that maps states to action values. |
| `train.py` | Optional command-line training entry point (equivalent to the notebook training loop). |
| `checkpoint.pth` | Saved weights of the successful agent. **Generated by training.** |
| `Report.md` | Description of the learning algorithm, hyperparameters, results, and future work. |

## How to Run

**Notebook (recommended):**
1. Open `Navigation.ipynb`.
2. Set `file_name` in cell 1 to your Banana environment path.
3. Run all cells. Training stops automatically once the 100-episode average reaches `+13` and saves `checkpoint.pth`. The score plot is saved to `assets/scores.png`.

**Command line (alternative):**
```bash
python train.py --env_path Banana.app          # trains, then saves checkpoint.pth + assets/scores.png
```

See [`Report.md`](Report.md) for the algorithm details and results.
